1、本示例是一个bae上的ibatis框架的demo，取自ibatis官方https://code.google.com/p/mybatis/downloads/detail?name=mybatis-jpetstore-6.0.1-bundle.zip&can=3&q=Product%3DSample
2、运行示例之前需要先创建bae应用，并在应用中创建mysql数据库，然后将src/database/ibatis.sql导入到数据库中 (创建mysql： http://developer.baidu.com/wiki/index.php?title=docs/cplat/rt/java/mysql#.E5.88.9B.E5.BB.BA)
3、修改src/mysql.properties文件中的数据库名、用户名和密码
4、提交代码运行即可
5、示例运行展示： http://ibatis.duapp.com/
